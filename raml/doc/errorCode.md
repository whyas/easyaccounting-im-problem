基本错误码：

| 错误码 | 解释    	|
|-------|-----------|
| 1000  | 成功返回 	|
| 1001  | 参数非法 	|
| 1002  | 资源不存在 	|
| 1003  | 操作暂未支持	|
| 1004  | 库房未找到 	|
| 9999  | 其它错误 	|
