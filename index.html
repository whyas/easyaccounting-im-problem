  • Response

Query Parameters

  • name: required (string)

    "itemPrice": 120,
    "totalAmount": 1200
  },
  {
    "itemId": 49,
    "itemCount": 10,
    "itemPrice": 120,
    "totalAmount": 1200
  }
]

    输出货物列表, JSON 数组，在 JSON 中传的时候不用封装成字符串
